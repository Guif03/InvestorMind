# InvestorMind
Transforme a complexidade do mercado financeiro em aprendizado simples. Este repositório utiliza o NotebookLM como um assistente inteligente, gerando insights, resumos e podcasts personalizados a partir de fontes confiáveis. O guia definitivo para quem quer começar a investir com segurança e inteligência artificial. 📈💼
